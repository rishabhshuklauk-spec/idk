## 🔗 How to open CSVs with working hyperlinks

GitHub displays CSVs, but **Excel is required** for clickable links.

### To open with working links:
1. Download the CSV  
2. Open Excel  
3. Go to **File → Open → Browse**  
4. Select the CSV  
5. Excel will automatically convert `=HYPERLINK("url","text")` into clickable links  

---

## 🛠 Editing the Data

You can edit CSVs directly in GitHub or locally in Excel.  
If editing in Excel, remember to **Save As → CSV UTF‑8** to preserve formatting.

---

## 📦 Combining into a single Excel workbook

If you want a full Excel workbook:

1. Open each CSV in Excel  
2. Copy/paste each into a new sheet  
3. Save as `Kitchen.xlsx`  

This preserves:
- Hyperlinks  
- Sheet structure  
- Formatting
